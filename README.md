# flowerpassword.js
Flower Password implementation for JavaScript.

## Installation

```
$ npm install xlsdg/flowerpassword.js
```


## Usage

Include the (minified) JavaScript script:

``` html
<script src="flowerpassword.min.js"></script>
```

The **fpCode** method:

``` html
var code = fpCode("password", "key", 16);
```
